## 📌 Sobre o projeto

Esta landing page foi criada com o objetivo de captar clientes para os serviços de:

- **Limpeza profissional de painéis solares**
- **Manutenção preventiva de sistemas fotovoltaicos**

Foco principal: residências, empresas, fazendas e comércios que já possuem energia solar.

## 🚀 Como ativar o site (GitHub Pages)

1. Vá em **Settings** do repositório

5. Clique em **Save**
6. Aguarde alguns minutos e acesse o link gerado
